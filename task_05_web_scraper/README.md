\# Web Scraper (Java)



\## Overview

This project is a simple web scraping program built using Java and the Jsoup library.  

The program extracts product information from an online e-commerce website and stores the data in a CSV file.



The scraper collects details such as product name, price, and rating.









\## Features

\- Connects to a live website

\- Extracts product information from HTML

\- Parses product name, price, and rating

\- Saves extracted data into a CSV file

\- Uses the Jsoup HTML parsing library









\## Technologies Used

\- Java

\- Maven

\- Jsoup

\- File I/O (CSV)









\## Website Used

Data is scraped from the practice website:



https://books.toscrape.com/



This site is designed for web scraping practice.









\## Output

The program generates a CSV file containing scraped data.



Example:



Name,Price,Rating  

A Light in the Attic,£51.77,Three  

Tipping the Velvet,£53.74,One  









\## How to Run

1\. Open the project in Eclipse / STS.

2\. Ensure Maven dependencies are downloaded.

3\. Run WebScraper.java.

4\. The scraped data will be saved as `products.csv`.
